Remote Desktop of Ubuntu 12.04

January 21, 2014

Install xRDP package:

$sudo apt-get install xrdp

If you have problem connecting to the desktop, run:

$echo "gnome-session --session=ubuntu-2d" > ~/.xsession $sudo apt-get install gnome-session-fallback
